§  STM32CubeF7 Firmware Package V1.16.1 / 12-February-2021

Main Changes

 

·         Updated hal_def.h to support new ARM compiler Keil V6.

·         Added new defines for ARM compiler V6

·         Updated HAL TimeBase TIM, RTC alarm and RTC WakeUp templates for more robustness.

·         ‘register’ keyword retired to be compliant with C++17 (already deprecated in C++11).

·         Added License.md and Readme.md files required for GitHub publication.
